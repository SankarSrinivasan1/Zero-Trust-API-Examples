# Zero-Trust-API-Examples
Practical Zero Trust API examples for real-world developers. Explore insecure vs secure implementations, JWT mistakes, and rate limiting strategies. Learn by breaking bad APIs and fixing them using authentication, validation, and least privilege principles. Built for hands-on security learning, not theory.

# Zero Trust API Examples

This repo is not here to impress you.

It is here to show how APIs actually break in real life.

Most developers think:
- My API works
- My auth is fine
- Internal services are safe

All three are usually wrong.

---

## What is inside

- bad-api-example → intentionally insecure API
- fixed-api-example → same API, but properly secured
- jwt-mistakes → common JWT errors developers make
- rate-limit-example → how to stop abuse properly

---

## How to use this repo

Do not just read.

- Run the bad example
- Break it
- Abuse it
- Then compare with the fixed version

That is how you learn security.

---

## Warning

Do not use the bad examples in production.

Seriously.
